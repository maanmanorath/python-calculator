# python-calculator
I am a beginner in the early stages of learning Python, and I built this command-line calculator to practice fundamental concepts and get better at managing program flow. The project relies on a while True loop to keep the program running, allowing the user to perform multiple calculations without having to restart the script.

My main goal was to learn how to manage loops and build a tool that handles large numbers more effectively than a standard calculator, which usually switches to scientific notation after 16 digits. To solve this, I imported the Decimal library and set the precision to 400 digits. My motive was to learn how to increase numerical accuracy and keep the output readable even with very large numbers.

During development, I ran into a bug that took up most of my time: I wasn't using the correct tab spacing for my break statement, which caused a syntax error. Fixing this taught me how important proper indentation is in Python. Overall, this project helped me learn how to import and use libraries, how to structure loops, and how to use the break command to control program execution.
